# Dark Side of Social Media – Statistical Data Analysis

## Project Overview

This university group project analyzed the impact of social media usage on user behavior, addiction levels, and productivity using a dataset of 1,000 observations.

## Objectives

* Analyze social media usage patterns
* Identify relationships between key variables
* Generate insights using statistical techniques

## Methods Used

* Exploratory Data Analysis (EDA)
* Correlation Analysis
* Regression Analysis
* Data Visualization

## Tools

* Microsoft Excel
